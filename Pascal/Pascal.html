Введение

 Для начала давайте познакомимся поближе с этим языком. Язык Pascal был создан Никлаусом Виртом в 1968—1969г. Этот язык является базой для всех других языков, поэтому его изучают в старших классах (с 8 по 11 класс) и на первых курсах вузов. На мой взгляд, Pascal является достаточно простым языком программирования и у вас не должно возникнуть проблен с его изучением. Вот такая запись в программе означает комментарий //комментарий

Структура программы

 Программа на Pascal состоит из

1)Название программы: тут вы должны назвать свою программу. Сначала пишем слово Program, а потом название. Пример - Program pascal; обязательно поставьте точку с зяпятой в конце! Название можете написать любое, кроме ключевых слов, которые употребляются в написании программы. Например, begin, if, for, while и т.д.

2)Раздел описания: В разделе описания вы можете прописать переменные, массивы, константы. О них мы поговорим дальше.

3)Тело программы: В теле программы заключается основной код.
Вся основная программа пишется между
Begin
//Код
End.

Program pascal; //название
var a,b,c :integer; //описание
Begin
//тело
End.

Переменные

 Переменные - это ечейки в которые вы помещаете какую-то информацию, а потом используете её. Переменные надо прописывать в разделе описания, между строками Program и Begin. Написание переменных начинается со слова var далее ставим пробел и пишем переменные. Если вы указываете более одной переменной, то ставьте между ними запятую! Нельзя писать переменные типа 1,2,3, brgin, if, for и т.д! Далее следует пробел, потом ставим двоеточие : и пишим тип данных. О них написано ниже. Итак, мы получаем вот такую картину.

program pascal; //низвание
var a,b,c :integer; //описание
begin
//тело
end.

Типы данных

 Основные типы данных - это integer (целочисленный тип), real (числа с плавущей точкой и целые), string (строки) и char (символы). Это самые важные и используемые типы данных. В дальнейшем будем использовать именно их.
Пример записи разных типов данных

1) var a :integer;
2) var a :real;
3) var a :string;
4) var a :char;

a - это переменная. Запись должна быть именно такой!

Операторы

 Самыми важными и распространёнными операторами в языке Pascal являются:

1) write или writeln (оператор вывода на экран какой-то информации пользователю) Разница между ними в окончании ln. Если добавить окончание ln, то информация будет выведина на следующую сточку, но это по вашему усмотрению. Например, write('Тут следующая информация не выведеца на другую строчку, а если добавить окончание ln, то переход на следующую строчку будет');


1) Если вы хотите вывести что-то так, как вы написали в программе, то в нутри круглых скобок добавьте две одинарные кавычки write('pascal');
2) Если вы хотите вывести значение переменной или какое-то ещё действие, то одинарные кавычки не нужны write(a + b);
3) Если вы хотите вывести текст и значение переменной, то между ними надо поставить запятую write('pascal',a);
Синтаксис: write('');.

2) read или readln (этот оператор позволит пользователю ввести иннформацию). Правило про ln тут такоеже, как и у write. Например, read(a);

3) := (оператор присваевания), = (оператор равно), * (оператор умножения), / (оператор деления), + (оператор сложения), - (оператор вычитания).
a := b + c;
a := b - c;
a := b * c;
a := b / c;

Условия

 Условие обозначаестя, как
if условие then
begin
что будет напечатано в результате выполнения условия
end
else (Это условие выполняется только в том случае если основное условие не выполняется)
Запомните, что пере else точка с запятой не ставится!

Program pascal;
var a,b :integer;
Begin
a := 1;
b := 2;
if a > b then
begin
write('Условие верно');
end //перед else точка с запятой не ставится
else
write('Основное условие не выполнилось');
End.

Обратите внимание на синтаксис!

Циклы

 Циклы - это
Основные циклы - это for и while. Структура цикла for:

for условие to условие do
begin
оператор
end;

Например:

Program pascal;
var a,b,c :integer;
Begin
b := 5;
for a := 1 to b do
begin
c := a + 1;
end;
write(a);
End.

Обратите внимание на синтаксис!
Структура цикла while:

while условие do
begin
оператор
end;

Например:

Program pascal;
var a,b :integer;
Begin
b := 5;
while b < 10 do
begin
a := a + 1;
b := b + 1;
end;
write(a);
End.

Обратите внимание на синтаксис!

Массивы

 Массивы - это набор элементов одного типа, каждый из которых имеет свой номер, называемый индексом (индексов может быть много, тогда массив называется многомерным). Каждый элемент массива описывается в общем виде а[i], где а[i] - значение элемента массива, а - имя массива, i - номер элемента в данном массиве а, который меняется от 1 до n.

Форма описания массива

var <имя массива> : array[<колличество элементов>] of <тип элементов>;

array - массив
<тип элементов> - например, integer (целочисленный тип).

Пример описание массива

Program pascal;
var a,i : array[1..10] of integer; //Раздел описания
Begin
End.

Обратите внимание на синтаксис!

Чтобы заполнит массив вам надо будет воспользоваться кодом, который представлен ниже.

Program pascal;
const Sz = 50; //Размер нашего массива
var
a : array [1..Sz] of integer;
b : integer;
i : integer;
begin
write('Введите количество элементов массива: ');
readln(b);
writeln('Введите элементы массива: ');
for i := 1 to b do
read(a[i]);
write('Вывод элементов: ');
for i := 1 to b do
write(a[i],' ');
end.

Ниже представлен код, с помощью которого можно обращатся к уже созданным элементам массива.

Program pascal;
var
a: array [1..5] of integer;
i: integer;
begin
write('Введите элементы: ');
for i := 1 to 5 do
read(a[i]);
write('Вывод элементов: ');
for i := 1 to 5 do
writeln(a[i],', ');
write('Значение заданного элемента массива = ',a[2]); //С помощью этой строчки можно узнать чему равен элемент массива
end.

Чтобы рандомно заполнить массив, используйте следующий код.

Program pascal;
var
a: array [1..10] of real;
i: integer;
begin
randomize;
for i := 1 to 10 do
begin
a[i] := random(100); //В этой строчке мы задаём предел рандомных чисел
write(a[i],', ');
end;
end.

Считаем

 В этой программе с помощью операторов +, -, *, / мы посчитаем переменные a и b, а переменная с будет выводить конечное значение на экран.

Program pascal;
var a,b,c :integer;
Begin
a := 5;
b := 5;
c := a + b; //Тут также могут быть операторы *, /, -
write('Конечное значение:',c);
end.

Пишем условие

 В этой программе с помощью условия if мы проверим a < b. Обратите внимание, что после if мы написали begin end; это надо для того, чтобы если не выполнится условие a < b, то программа пропускала этот кусок кода и шла дальше!

Program pascal;
var a,b,c :integer; //У нас будут 3 переменные a,b,c целочисленного типа
Begin
a := 10;
b := 20;
if a < b then //Если a < b то
begin
c := a + b;
write('Значение с = ',c);
end;
End.

Условие с else

 Здесь мы используем if else. У нас условие a < b не выполнится, а else выведет своё значение. Запомниет, что перед else точка с запятой не ставится.

Program pascal;
var a,b,c :integer; //У нас будут 3 переменные a,b,c целочисленного типа
Begin
a := 20;
b := 10;
if a < b then //Если a < b то
begin
c := a + b;
write('Значение с = ',c);
end //Перед else точка с запятой не ставится!
else
write('Условие a < b не выполнилось. Сработало else');
End.

Пишем цикл for

 Сейчас мы разберём программу с цыклом for. Задание: вычислить, чему будет равно c если значение b = 5.

Program pascal;
var a,b,c :integer;
Begin
write('Введите число: ');
read(b); //это число будет равно 5
for a := 1 to b do
begin
c := c + a;
end;
write('Число c = ',c);
End.

Программа выведет 15 при условии, что b = 5.

Пишем цикл while

 В этой программе мы воспользуемся циклом while. Задача: с помощью цикла while посчитать чему будет равно значение переменной с, если значение a = 5, значение b = 0.

Program pascal;
var a,b,c :integer;
Begin
a := 5;
b := 0;
while a > 1 do //пока а больше 0 выполнять
begin
a := a - 1;
b := b + 5;
c := b;
end;
write('Значение c = ',c);
End.

В результате программа выведет 20.

Первая программа на уровне ОГЭ

 Задача 20.2: Напишите программу, которая в последовательности натуральных чисел определяет максимальное число, кратное 5. Программа получает на вход количество чисел в последовательности, а затем сами числа. В последовательности всегда имеется число, кратное 5. Количество чисел не превышает 1000. Введённые числа не превышают 30 000. Программа должна вывести одно число — максимальное число, кратное 5.
Задача взята из сайта решу огэ: https://inf-oge.sdamgia.ru/test?id=3693517
Задачи уровня ОГЭ можно посмотреть здесь.
Решение:

Program pascal;
var
max,i,n,y :integer;
Begin
max:= -30001;
writeln('Введите кол-во чисел:');
read(n);
for i:=1 to n do
begin
writeln('Введите число');
read(y);
if ((y > max) and (y mod 5 = 0)) then
begin
max:=y;
end;
end;
writeln('max = ',max);
End.
