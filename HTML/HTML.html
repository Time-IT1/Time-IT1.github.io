Введение

 Для начала давайте познакомимся поближе с HTML. Язык HTML был создан британским учёным Тимом Бернерсом-Ли примерно в 1986—1991 годах. HTML создавался с целью обмена научной и технической документацией для людей не являющимися специалистами в области вёрстки. С помощью HTML можно легко создать простой, но красиво оформленный сайт, который в будующем сможет приносить вам доход. Но если вы хотите создать понастоящему красивый и привлекательный сайт, то вам обязательно нужно изучить CSS.

Подготовка

 Чтобы приступить к написанию кода вам надо скачать текстовый редактор notepad++ вот ссылка на скачивание. После того, как перейдёте по ссылке, спуститесь чуть-чуть ниже и нажмите на кнопку, которая на картинке обведена зелёной рамкой. Думаю, что объяснять вам установку не стоит.

1) После того, как вы всё установили, нужно создать папку и назвать её так, как будет называться ваш будущий сайт.
2) Зайдите в папку и там создайте, нажав правую кнопку мыши и ыбрав пункт создать, текстовый документ.
3) После, переименуйте фаил в название сайта и измените расширение с .txt на .html
4) После этой процедуры, откройте текстовый редактор notepad++.
5) В левом верхнем углу нажмите на кнопку фаил.
6) Выберите пункт - новый.
У вас появился новый документ. В нём мы и будем записывать код.
7) Чтобы сохранить документ вам надо в левом верхнем углу нажать на кнопку фаил.
8) Нажмите на Сохранить как... и выберите тот документ, который вы создали в 3) пункте.

Структура html

 Структура html документа всегда одинакова. Сейчас я покажу элементы, которые не изменяются. Чтобы их лучше запомнить запишите в свой созданный документ. Эту структуру можно представить, как скелет. Именно с ней мы и будем работать.

<!DOCTYPE html>
<html>
<head>
<meta http-equiv="Content-Type" content="text/html; charset=UTF-8"/>
<title>Изучение html</title>
</head>
<body>

</body>
</html>

Теги и атрибуты

 Итак, теги и атрибуты - это то, из чего состоит весь документ html. Теги могут, например, делать текст жирным, большим, маленьким шрифтом, также могут добавлять на сайт картинки, ссылки и многое другое. Атрибуты позволяют расширить возможности тегов. Например, возьмём тег <i></i> (он делает текст курсивным) и запишем в него слово text. К любому тегу можно приписать атрибут title. Этот атрибут позволяет, при наведении и удержании мышки на элементе, показывать вспомогательный текст. Вот им и воспользуемся. Общий вид тегов с атрибутами такой <тег атрибут="значение атрибута">текст</тег> (пока не задавайтесь вопросом, зачем ставить </тег>. Далее мы всё это разберём). Наш тег <i></i> с атрибутом title="" будет иметь следующий вид <i title="Это текст курсивом">text</i>

Вот пример: text.

Ещё можно писать тег внутри тега. Это всё будет выглядеть таким образом <i><b>Этот текст курсивный и жирный</b></i> (тег <b></b> делает текст жирным).

Сейчас я отвечу на вопрос, почему нужно в конце написания тегов ставить </тег>. Различают два типа тегов 1) парные и 2) непарные теги. Парные, например, тег <b></b>, <i></i> и другие. Непарные, например, <br/> (этот тег отвечает за перевод строки), <hr/> (этот тег добавляет горизонтальную линию там, где вы его поставили) и другие.

Список самых употребляемых парных тегов (по моему мнению).

1) <p></p> (создаёт параграф)
2) <b></b> (жирный шрифт)
3) <i></i> (курсивный шрифт)
4) <span></span> (определяет строчный элемент документа)
5) <div></div> (блочный тег)
6) <h1></h1> (Размер заголовков от самого большого h1 до самого маленького h6)
7) <ul></ul> (создаёт маркированный список (внутри этого тега обязательно должен быть тег <li></li>))
8) <ol></ol> (создаёт нумерованный список (внутри этого тега обязательно должен быть тег <li></li>))
9) <li></li> (действует внутри тегов <ul></ul> и <ol></ol>)
10) <button></button> (создаёт кнопку)
11) <center></center> (ставит контент в центер сайта)
12) <header></header> (создаёт шапку сайта)
13) <footer></footer> (создаёт подвал сайта)
14) <q></q> (обозначает цитату)
15) <u></u> (подчёркивает текст)

Теперь список самых употребляемых непарные тегов (по моему мнению).

1) <br/> (отвечает за перевод строки)
2) <hr/> (добавляет горизонтальную линию там, где вы его поставили)
3) <img/> (добавляет картинку)

Сейчс мы изучим основные атрибуты к этим тегам. Но сначала посмотрите, как выглядит атрибут внутри тега <тег атрибут="значение атрибута"></тег>. Атрибут записывается так и в парных и в непарный тегах (в непарных записывается так <тег атрибут="значение атрибута"/>). Перейдём к таблице с основными атрибутами.

1) title (он подходит для любого тега и выполняет функцию подсказки, например, text)
2) style (он тоже подходит для любого тега и выполняет функцию, которая позволяет записать стили к данному элементу не переходя в css документ)
3) class (этот атрибут подойдёт к любому тегу. С помощью этого атрбута вы можете задать имя элементу, которое можете присваивать любым другим элементам, а также прописывать в css коде и добавлять стили, но это уже будет разбираться на курсе по css)
4) id (этот атрибут подойдёт к любому тегу. С помощью него вы сможете задать определённое имя одному или двум элементам, а потом использовать в css документе(вот чем отличается id от class (id с определённым именем можно использовать только на два элемента, а class на сколько хочешь)))
5) href (этот атрибут используется только с тегом <a></a> и его значение - это ссылка)
6) name ()
7) src (этот атрибут используется только с тегом <img/> и его значение будет ссылка или локальный путь к картинке)

head сайта

 Head - это раздел, который не виден пользователю. В нём прописываются разные meta теги и подключаются разные плагины или стороннии ресурсы, например css, javascript, разные шрифты и многое другое.

Эта часть кода, кроме <title></title>, не видна пользователю, но она очень важна для поисковых роботов.

Сейчас вы узнаете, что такое meta, link и т.д

meta - это тег предназначенны для тго, чтобы хранить информацию для браузера и поисковых систем.
link - это тег, который подключает сторонние файлы.

1) <meta http-equiv="Content-Type" content="text/html; charset=UTF-8"/> (эту строчку просто запишите себе и, когда будите писать head, то просто вставляйте его и всё. А вообще она определяет какую кодировку вы используете)
2) <meta name="keywords" content="" /> (эта строчка нужна для поисковых роботов. В ней вы должны указать ключевые слова на своей странице (это, как хэштэги))
3) <meta name="description" content="Краткое описание" /> (в этой строчке вы должны написать краткое описание вашей страници)
4) <link href="Путь к css файлу" type="text/css" /> (эта строчка подключает css фаил к html документу. Путь указывается с помощью атрибута href)
5) <link href="Путь к картинке" rel="shortcut icon" type="image/x-icon" /> (с помощью этой строчки можно поставить картинку на закладке сайта. Путь указывается с помощью атрибута href)
6) <script type="text/javascript" src="Путь к JavaScript файлу"></script> (эта строчка позволит подключить JavaScript файл)
7) <meta name="viewport" content="width=device-width, initial-scale=1.0"> (при адаптировании сайта вам понадобится вот эта строчка)

Вот, как выглядит description. Он обведён в зелёный цвет

А название сайта - это title.

Вся эта информация прописывается в теге <head></head>.

body сайта

 Body - это тело сайта. В нём прописывается весь код, который отображается на сайте в виде того, что вы написали. Этот код прописывается между тегом <body></body>.

Body можно разделить на три части:

1) <header></header> (Это "шапка" сайта. Там находится название, меню и т.д)
2) <content></content> (Это центральная часть сайта. В ней распологается основная информация)
3) <footer></footer> (Это "подвал" сайта. В этой части располагается copyright и т.д)

О них мы сейчас и поговарим.

Если смотреть на этот сайт, то header это самый верх сайта, там распологается наше название и меню.
